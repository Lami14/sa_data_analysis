# South Africa Socio-Economic Insights Dashboard

This project analyzes publicly available South African data on crime, unemployment, and load-shedding. The goal is to provide interactive insights and visualizations for socio-economic trends.

## Features
- Crime trends per province
- Load-shedding trends
- Unemployment heatmaps
- Streamlit interactive dashboard

## Data Sources
- [Stats SA](http://www.statssa.gov.za)
- [Eskom Load Shedding](https://loadshedding.eskom.co.za)
- [SAPS Crime Stats](https://www.saps.gov.za/services/crimestats.php)

## Stack
- Python, Pandas
- Seaborn & Plotly
- Streamlit
- Jupyter Notebook

## Run Dashboard
```bash
pip install -r requirements.txt
streamlit run app.py
